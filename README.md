# excel-Perlerbeads-Simulator
使用豆包+Codebuddy+DeepSeek实现的一款excel拼豆模拟器，仅使用了HTML、CSS、JS技术。上传你喜欢的图案生成对应的excel表格拼豆图案，简单易上手，有无技术经验均可使用
# 使用教程
下载或拉取源代码后，在浏览器运行html文件即可

# 规则
##技术栈
1. 基础页面框架
HTML5：页面结构、上传控件、预览容器、按钮、参数表单
Tailwind CSS v3：全部页面样式、响应式布局，简洁浅色 UI
JavaScript ES6+：所有业务逻辑
2. 图片处理核心库（本地浏览器处理图片，不上传服务器）
Canvas API：图片像素读取、缩放、网格绘制、预览渲染
color-thief-browser：图片取色、色彩聚类、自动减色，生成纯色块
html-to-image（可选备用）：仅网格预览渲染辅助，非必需
3. Excel 生成导出核心库（SheetJS/xlsx）
SheetJS (xlsx.full.min.js)：
创建多工作表
设置单元格背景色
写入单元格文本坐标 (A1/B2)
统一行列宽高实现正方形格子
浏览器直接下载.xlsx 文件，无需服务器
4. 工具辅助库
LocalStorage：缓存用户上次参数（网格宽高、减色数量）
原生 FileReader：读取本地图片文件，不经过后端
防抖函数工具：调整参数时延迟重绘预览，防止页面卡顿
5. 浏览器兼容要求
支持 Chrome / Edge / Firefox 最新版；不兼容 IE；页面顶部增加浏览器兼容提示。
6. 页面运行规则
所有图片解析、像素拆分、Excel 生成全部在用户本地浏览器完成，图片不会上传任何服务器，页面显眼处标注隐私说明。


##Excel 坐标换算规则
列索引 0=A，1=B，2=C……25=Z，26=AA，27=AB，以此类推，严格匹配 Excel 原生列命名
行索引 0 = 行 1，索引 1 = 行 2，一一对应
色彩同步规范
网页预览色块 RGB 与 Excel 单元格填充 RGB 完全一致，无色差
数据同源规范

# 功能清单&&提示语
我现在想做一款Excel 拼豆模拟器网页版 V1.0，以下是功能清单
# 一、页面整体布局结构（固定结构，AI严格遵循）
页面分为4大区域，从上至下排布：
1. 顶部标题栏：项目名称、隐私提示文字
2. 左侧操作控制面板（固定左栏）：图片上传区、尺寸参数、减色参数、操作按钮
3. 中央预览区：原图预览 | 拼豆网格预览（双栏并排）
4. 底部功能按钮区：【生成并下载Excel】主按钮

# 二、分模块详细功能描述清单（V1.0限定需求）
## 模块1：图片上传模块
### 功能1-1 图片导入方式
1. 支持点击上传框选择本地图片；支持拖拽图片放入上传框
2. 支持图片格式：jpg、jpeg、png、bmp
3. 文件大小限制：单张图片最大5MB，超出弹出提示“图片过大，请压缩后重试”
4. 无图片时，上传框显示灰色占位提示文字：「点击或拖拽上传图片」

### 功能1-2 原图实时预览
1. 上传图片后，左侧预览框自动渲染原图，等比例缩放适配容器
2. 支持清空图片按钮：一键移除当前图片，重置所有参数与预览网格
3. 未上传图片时，网格预览区域显示空白提示文字：「请先上传图片」

## 模块2：网格尺寸参数配置模块
### 功能2-1 网格行列设置
1. 两组输入框：网格宽度（列数）、网格高度（行数）
2. 内置快捷预设按钮：10×10、16×16、24×24、32×32
   - 点击预设自动填充宽、高输入框并实时刷新网格预览
3. 数值限制：最小8，最大60；输入超出范围自动修正为边界值并弹窗提示
4. 输入数字实时生效，修改后自动重新拆分图片色块、刷新预览网格

### 功能2-2 图片像素缩放逻辑
1. 根据设置的宽高，自动将原图压缩为「宽列数 × 高行数」像素矩阵
2. 采用就近采样算法，把原图每个区域映射为单一色块，消除渐变过渡

## 模块3：色彩减色拆分模块（核心色块拆分逻辑）
### 功能3-1 最大颜色数量设置
1. 输入框：最大用色数量，范围2~64
2. 修改数值后自动对像素矩阵做色彩聚类合并，相近颜色统一为一个色块
3. 聚类规则：视觉相近RGB自动合并，减少杂色，适配拼豆纯色需求

### 功能3-2 色块数据存储
1. 处理后生成二维数组 gridData[行][列]
   单格数据结构示例：
   ```
   {
     x: 列序号(0开始),
     y: 行序号(0开始),
     excelCol: "A", // 对应Excel列名 A/B/C...
     excelRow: "1", // 对应Excel行号 1/2/3...
     excelCoord: "A1", // 完整坐标
     rgb: "rgb(255,255,255)", // 色块填充色
   }
   ```
2. 该二维数组同时供给【网页网格预览】和【Excel生成】共用一套数据，保证坐标、颜色完全同步无偏差

## 模块4：网页网格预览&坐标展示模块
### 功能4-1 网格画布渲染
1. 根据gridData二维数组渲染正方形网格，一格对应一颗拼豆
2. 单元格背景填充对应rgb色块，网格添加浅灰色分割边框，区分格子

### 功能4-2 Excel坐标双展示（固定两种模式，V1.0全部实现）
#### 模式1：图纸模式（默认开启）
1. 网格顶部标注列字母 A、B、C、D……
2. 网格左侧标注行数字 1、2、3、4……
3. 格子内部无文字，只显示纯色块，打印清晰

#### 模式2：明细坐标模式
1. 顶部、左侧依旧保留行列标注
2. 每个格子内部居中小字写入完整坐标（如A1、B5）
3. 增加切换开关，一键切换两种预览样式，切换实时刷新画布

### 功能4-3 鼠标悬浮提示
鼠标悬停任意网格格子，弹出悬浮气泡，显示内容：
完整坐标：A1 | RGB色值：rgb(xxx,xxx,xxx)

## 模块5：Excel文件生成与下载（核心输出功能）
点击底部【生成并下载Excel】按钮执行整套逻辑，自动生成.xlsx并触发浏览器下载，文件名格式：拼豆图纸_年月日.xlsx
自动生成2个工作表（V1.0只做两张，第三张明细表延后迭代）
### Sheet1：拼豆主图纸（核心表）
1. 行列范围严格匹配设置的网格宽高
2. 统一设置所有单元格行高=列宽，强制正方形格子
3. 每个单元格背景填充对应色块RGB
4. 坐标遵循当前选择的模式：
   - 图纸模式：单元格空白，仅依靠Excel自带A/B列标、1/2行号识别坐标
   - 明细模式：单元格内写入文本“A1”居中，字体小号不遮挡色块
5. 全部单元格添加细灰色边框，方便打印分割

### Sheet2：颜色统计清单
1. 表格包含三列：颜色色块、RGB色值、该颜色总豆子数量
2. 自动汇总gridData内所有颜色，统计每种色块出现次数
3. 表格顶部增加总数量汇总行：拼豆总颗数：XXX

## 模块6 全局辅助基础功能
### 功能6-1 参数本地缓存
使用LocalStorage保存用户上次操作参数：网格宽、网格高、最大颜色数、坐标模式；页面刷新后自动读取加载，无需重复设置

### 功能6-2 加载状态提示
1. 图片解析、色块运算、生成Excel过程中显示加载动画+文字提示
2. 运算完成自动关闭加载弹窗

### 功能6-3 异常拦截提示
1. 未上传图片时点击下载按钮：弹窗提示「请先上传图片再生成图纸」
2. 图片解析失败：提示「图片解析出错，请更换图片重试」
3. 网格尺寸超过60限制：提示「最大支持60×60网格，请缩小尺寸」


网页预览网格、Excel 图纸读取同一套 gridData 二维数组，保证坐标一一对应，不会出现网页和 Excel 坐标错位
UI 规范
整体简约白色界面，主按钮蓝色，操作清晰无多余装饰，无弹窗广告、无多余营销模块
